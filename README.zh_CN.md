# egg-wechat-pay

node-wechat-pay Egg.js 插件。

## 安装插件

```bash
npm i @axolo/egg-wechat-pay --save
```

## 开启插件

```js
// config/plugin.js
exports.wechatPay = {
  enable: true,
  package: '@axolo/egg-wechat-pay',
};
```

## 详细配置

请到 [config/config.default.js](config/config.default.js) 查看详细配置项说明。

```js
// {app_root}/config/config.default.js
exports.wechatPay = {
};
```
